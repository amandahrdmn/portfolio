Academy Portal

This project was a fullstack legacy application designed to enable staff at the Academy to register and keep track of academy applicants and hiring partners as well as create events for interaction between the two groups. Mayden Academy Aptitude Test

This project involved modifying a pre-existing Javascript-based application designed to test the aptitude of prospective Mayden Academy applicants. This involved using asynchronous fetch requests to a pre-built API as well as HTML templates to populate both an online test for applicants as well as an admin page for Mayden Academy staff members.

Armadillo's Numbers Game

This project was a point-and-click numbers game, where players were given points for clicking sets of randomly generated, increasingly difficult-to-distinguish numbers in the correct order within a specified time limit. We used asynchronous timers for the countdown and custom-built event listeners to handle the click-dependent DOM manipulation.
